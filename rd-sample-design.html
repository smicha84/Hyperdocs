Sample Archetypes
No single session has all these characteristics. The R&D sample needs multiple sessions that collectively cover every output field. 6 archetypes identified.

A: The Full Lifecycle

500+ messages • covers almost everything

File creation, modification, debugging, testing, frustration, resolution, pivots, breakthroughs. The full emotional and technical arc.

Exercises: all action_vectors, all emotional_tenors, all confidence_signals, micro+meso+macro geological, 20+ idea nodes, all 10 edge types, all marker categories, file dossiers with warnings

B: The Frustration Session

100-300 messages • emotional intensity

Heavy user frustration, CAPS, profanity, emergency interventions, repeated corrections of Claude. The human-AI friction record.

Exercises: frustration_peaks[], emergency_interventions, emotional_tenor=frustrated, reactions thread, Claude behavior patterns, friction_log

C: The Architecture Session

200-500 messages • system design focus

System design decisions, file creation, dependency mapping, pattern discussions. Calm, productive, decision-heavy.

Exercises: threads.software, threads.code, action_vector=created/decided, grounded_markers architecture+decision, decision_trace

D: The Debugging Session

50-150 messages • error-focused

Error traces, failed tool calls, bisecting bugs, eventual fix. The confidence arc from fragile to proven.

Exercises: error_count, tool_failure_count, action_vector=debugged, confidence=fragile→proven, intent_marker=bugfix, emotional_tenor=uncertain→relieved

E: The Subagent Session

20-80 messages • spawned task

Spawned by orchestrator, focused task, rapid completion. Narrow scope, single-file or single-purpose analysis.

Exercises: is_subagent=true, agent_type, narrow file set, focused dossier

F: The Cross-Session Reference

any length • continuity-focused

Explicit references to prior sessions, resuming work, context recovery. "Continuing from last session" patterns.

Exercises: cross_session_links[], session continuation, edges with "resurrected", char_per_line encoding artifacts
